# 内容踩坑记录

## 2026-03-11 午间推文

### 问题：内容太长，重要信息被截断

**原文**：约 180 字符，但包含换行和 emoji，实际显示超 280 字符

**结果**：第 1 点（最重要的技巧）在时间线预览中被截掉

### 教训
1. **X 推文 280 字符限制**：包含换行、emoji 都算字符
2. **预览更重要**：用户刷到时只看前 2-3 行
3. **重要信息前置**：最有价值的放最前面

### 解决方案
- 推文控制在 150 字符以内（留余量）
- 或者用"全文在评论区"引导点击
- 或者发推文串（Thread）
