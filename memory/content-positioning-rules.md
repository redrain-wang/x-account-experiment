# 内容定位规范（2026-03-13 补充）

## ⚠️ 今天的问题

### 事件：想发车载手机支架求助

**原文**：
```
打算买一个车载手机支架，不想用我的夹在空调出风口上面的这种，大家有推荐吗。
```

### 问题分析

| 问题 | 说明 |
|------|------|
| **与账号定位不符** | 账号是"AI 与开发"，不是生活好物推荐 |
| **粉丝预期错位** | 粉丝关注是为了学 AI，不是看数码配件 |
| **内容调性断层** | 之前发 AI 工具/技术科普，突然发车载支架 |
| **像营销号** | 容易被误会是软广告 |

### 教训

1. **保持账号定位纯粹** - 不发与 AI/技术无关的内容
2. **生活内容发私人号** - 个人生活和工作账号分开
3. **如果一定要发** - 结合技术人视角或 AI 工具类比

## ✅ 正确的做法

### 方案 A：不发（推荐）
- 保持账号定位纯粹
- 生活内容发私人号
- 专注 AI/技术干货

### 方案 B：技术人视角（如果一定要发）
```
🚗 选个车载支架比选 AI 工具还难...

研究了一下午：
• 重力联动 vs 磁吸
• 出风口夹 vs 胶粘底座
• 单手取放 vs 双手操作

最后发现：没有完美的，只有合适的。

跟选 AI 工具一个道理。

大家用的什么支架？求推荐。
#数码配件 #技术人的纠结
```

**优点**：
- ✅ 有技术人思维（对比分析）
- ✅ 结合 AI 工具类比（呼应定位）
- ✅ 有观点（"没有完美的，只有合适的"）

## 📋 账号定位边界

### ✅ 可以发
| 类型 | 示例 |
|------|------|
| AI 工具推荐 | 秘塔写作猫、Gamma、Perplexity |
| 技术科普 | RAG 解释、技术概念通俗化 |
| 效率技巧 | AI 用法、工作流优化 |
| GitHub 热门 | 开源项目推荐 |
| AI 前沿新闻 | ChatGPT 新功能（需验证时效性） |
| 搞钱实验进展 | 粉丝数据、踩坑记录 |
| 技术人日常 | 开发日常、AI 心得 |

### ❌ 不建议发
| 类型 | 示例 | 原因 |
|------|------|------|
| 生活好物 | 车载支架、厨房用品 | 与定位无关 |
| 纯广告 | 产品推广、软广 | 损伤信任 |
| 政治敏感 | 时事评论 | 风险高 |
| 负能量 | 抱怨、吐槽 | 影响人设 |
| 互粉引流 | "互粉必回" | 粉丝质量差 |

## 💡 判断标准

发之前问自己：
1. **这条内容和 AI/技术有关吗？**
2. **粉丝看了会有收获吗？**
3. **像真人分享还是营销号？**
4. **符合"宇哥 | AI 与开发"的人设吗？**

**有任何一个是"否"，就不发。**
