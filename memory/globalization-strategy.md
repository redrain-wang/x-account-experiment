# X 账号全球化战略（2026-03-14 起）

## 🌍 语言策略

| 项目 | 决策 |
|------|------|
| **语言比例** | 80% 英文 + 20% 中文 |
| **目标市场** | 全球（主要是英语国家） |
| **老粉丝** | 不重要，专注新受众 |
| **变现方向** | 美元付费、全球广告、联盟佣金 |

## 📅 内容安排

| 时间 | 内容类型 | 语言 | 比例 |
|------|---------|------|------|
| **9:00** | AI Tool Review | 🇺🇸 英文 | 80% |
| **12:00** | AI News | 🇺🇸 英文 | 80% |
| **20:00** | 技术科普/心得 | 🇨🇳 中文 | 20% |

## ✍️ 英文写作原则

| 原则 | 说明 | 示例 |
|------|------|------|
| **用短句** | 不用复杂从句 | "AI drafts, I edit" |
| **用口语** | 像朋友聊天 | "Work smarter, not harder" |
| **有观点** | 敢说"我觉得" | "The goal isn't..." |
| **有互动** | 引导评论 | "How do you use AI?" |
| **避免机翻** | 不用中式英语 | ❌ "AI can help you improve efficiency" |

## 📋 内容生成规范

### 英文推文 + 中文翻译
- ✅ **每条英文推文都必须附带中文翻译**
- ✅ **翻译要通俗易懂**（雨哥是"小学生"😄）
- ✅ **格式清晰**：【英文原文】+【中文翻译】
- ✅ **方便雨哥理解** 发的是什么内容

## 🎯 为什么选英文？

| 优势 | 说明 |
|------|------|
| **市场更大** | 全球用户，不只是中文圈 |
| **变现机会多** | 美元付费、全球广告 |
| **竞争相对小** | 中文 AI 内容卷，英文还有机会 |
| **单价高** | 美元广告分成 > 人民币 |

## ⚠️ 挑战

| 挑战 | 解决方案 |
|------|---------|
| **语言门槛** | 用地道短句，不用复杂表达 |
| **文化差异** | 学习英文社区表达方式 |
| **竞争也大** | 专注垂直领域（AI 工具 + 效率） |
| **老粉丝流失** | 不重要，专注新受众 |
